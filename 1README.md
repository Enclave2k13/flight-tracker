# Проект flight-tracker

## Описание:

Проект flight-tracker - это консольное приложение на Python для отслеживания самолетов в реальном времени. Программа получает данные о воздушных судах через OpenSky API, отображает информацию о рейсах и сохраняет её в JSON-файл.

## Возможности:

- Получение данных о самолетах над любой страной мира
- Отображение информации: позывной, страна регистрации, высота, скорость, статус (в воздухе/на земле)
- Сохранение данных в JSON-файл без дубликатов
- Фильтрация самолетов по высоте и стране регистрации
- Статистика по сохраненным данным
- Цветной консольный интерфейс

## Установка:

1. Клонируйте репозиторий:
```
git clone git@github.com:Enclave2k13/flight-tracker.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
## Использование:

1. Запустите приложение:
```
python src/main.py
```
2. В главном меню выберите действие:
   - **1** - получить данные о самолетах над страной (например, France)
   - **2** - показать топ N самолетов по высоте
   - **3** - найти самолеты по стране регистрации
   - **4** - показать статистику по сохраненным данным
   - **0** - выход

## Тестирование:

Проект включает unit-тесты с использованием pytest. Для запуска тестов:

```
pytest tests/ -v # Обычный запуск тестов
pytest --cov=src --cov-report=html tests/ # С отчетом о покрытии
start htmlcov/index.html  # Открыть отчет в браузере
```
