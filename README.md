# Проект flight-tracker

## Описание:

Проект flight-tracker - это консольное приложение на Python для отслеживания самолетов в реальном времени. Программа получает данные о воздушных судах через OpenSky API, отображает информацию о рейсах и сохраняет её в JSON-файл.

## Возможности:

- Получение данных о самолетах над любой страной мира
- Отображение информации: позывной, страна регистрации, высота, скорость, статус (в воздухе/на земле)
- Сохранение данных в JSON-файл без дубликатов
- Фильтрация самолетов по высоте и стране регистрации
- Статистика по сохраненным данным
- Цветной консольный интерфейс
